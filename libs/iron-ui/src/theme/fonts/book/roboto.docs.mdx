### Overview

This component envelope the signature "CircularPro book" font of Watheia projects.

```tsx
import { bookFont } from '@watheia/iron-ui.theme.fonts.book';

function ClientContext({ children }) {
  return (
    <div className={bookFont}>
      {children}
      <div>I am text that will appear in CircularPro Book font!</div>
    </div>
  );
}
```
